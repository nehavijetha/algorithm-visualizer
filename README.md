# Algorithm Visualizer — Sorting & Searching

A simple interactive visualizer for:
- Bubble Sort
- Merge Sort
- Binary Search

Built using **pure HTML + CSS + JavaScript**  
Deployed on **GitHub Pages**.

## How to run
Just open `index.html` in any browser.

## Deployment
Go to GitHub → Settings → Pages → Select:
- **Branch:** main  
- **Folder:** /(root)

Your site will go live in under 1 minute.

## What I learned
- How sorting algorithms work internally  
- How to visualise comparisons, swaps, and pointer movements  
- How to build and deploy static JS apps
